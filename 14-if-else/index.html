Estruturas Condicionais if | else

As estruturas condicionais permitem que o programa tome decisões com base em condições. Elas controlam o fluxo da aplicação, tornando o código mais inteligente e dinâmico.

  • O JavaScript avalia a condição.
  • Se for true, executa o bloco do if.
  • Se for false, executa o bloco do else.
